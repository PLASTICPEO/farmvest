## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/PLASTICPEO/farmvest.git
   ```
2. Install packages
   ```sh
   yarn
   ```
3. Start
   ```sh
   yarn dev
   ```
